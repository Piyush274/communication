OS-CFAR vs CA-CFAR: CA-CFAR uses the mean of all reference cells as threshold. When multiple strong interferers sit inside the reference window, they inflate the mean → threshold spikes → real targets are masked. OS-CFAR sorts reference cells and picks the k-th order statistic, effectively discarding the top (N−k) outlier cells (likely interferers). Result: a stable threshold even in a dense interference environment, with far fewer missed detections.
